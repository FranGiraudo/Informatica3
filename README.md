Prácticos - Informática 3

Este repositorio contiene la colección completa de ejercicios y estructuras de datos implementadas en Java a lo largo del semestre, cubriendo los siguientes temas:

* **Programación Orientada a Objetos (POO)**
* **Recursividad**
* **Estructuras Lineales (Pilas, Colas, Listas Enlazadas)**
* **Árboles de Búsqueda (AVL y Rojinegro)**
* **Montículos Binarios y Colas de Prioridad**
* **Algoritmos de Ordenamiento**
